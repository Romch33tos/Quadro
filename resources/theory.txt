1. Понятие квадратного уравнения

Квадратным уравнением называют уравнение вида:

ax² + bx + c = 0, 

где а, b и c – это числа, называемые коэффициентами и при этом, коэффициент a ≠ 0. 

Коэффициент а называют старшим коэффициентом, коэффициент b – вторым коэффициентом, а коэффициент с – свободным членом. 

Если коэффициент a равен единице, то говорят, что квадратное уравнение является приведенным. 

Квадратные уравнения делятся на полные и неполные. 

Полное квадратное уравнение – это квадратное уравнение, коэффициенты b и c которого отличны от нуля. 
Если же один из этих коэффициентов равен нулю, то такое уравнение называют неполным. 

Существует три вида неполных квадратных уравнений:

ax² + c = 0, при b = 0;
ax² + bx = 0, при c = 0;
ax² = 0, при b = 0 и c = 0.

Решить квадратное уравнение – значит найти его корни, значения переменной x, при которых уравнение превращается в верное равенство или доказать, что их не существует.

2. Понятие дискриминанта

Дискриминант – это выражение, по знаку которого можно определить количество корней уравнения. 

Выделяют три следующих случая:

Если D < 0, то уравнение не имеет корней;
Если D = 0, то уравнение имеет два одинаковых корня;
Если D > 0, то уравнение имеет два различных корня.

Формула дискриминанта: D = b² - 4ac
Формулы корней: 
x₁ = (-b + √D) / 2a; 
x₂ = (-b - √D) / 2a. 

Пример решения уравнения через дискриминант: 

x² - 6x + 5 = 0
D = -6² - 4 · 1 · 5 = 36 - 20 = 16
x₁ = (6 + √16) / (2 · 1) = (6 + 4) / 2 = 10 / 2 = 5
x₂ = (6 - √16) / (2 · 1) = (6 - 4) / 2 = 2 / 2 = 1

3. Метод решения через половину коэффициента b

В случае если коэффициент b – четное число, то формулу дискриминанта можно упростить, введя переменную k, равную половине коэффициента b. Такой дискриминант обозначают как  D/4, так как он составляет ровно четверть от изначального значения.

Формула дискриминанта: k = b/2; D/4 = k² - ac.
Формулы корней:
x₁ = (-k + √(D/4)) / a; 
x₂ = (-k - √(D/4)) / a. 

Пример решения уравнения через половину коэффицента: 

x² - 6x + 5 = 0
k = -3, D = -3² - 1 · 5 = 9 - 5 = 4
x₁ = (3 + √4) / 1 = (3 + 2) / 1 = 5
x₂ = (3 - √4) / 1 = (3 - 2) / 1 = 1

Помимо формул для дискриминанта, существуют и другие способы решения, которые позволяют упростить процесс вычисления корней квадратного уравнения. 

4. Теорема Виета

Теорема Виета позволяет решать приведённые квадратные уравнения путем подбора корней.

Формулировка теоремы: сумма корней уравнения равна второму коэффициенту с противоположным знаком, а произведение корней равняется свободному члену. 

Это условие можно записать в виде системы двух уравнений. Числа, удовлетворяющие её условиям, являются корнями уравнения.

Система уравнений для подбора корней:
x₁ + x₂ = -b
x₁ ∙ x₂ = c

Пример решения уравнения через теорему Виета:

x² - 7x + 12 = 0
x₁ + x₂ = 7    
x₁ ∙ x₂ = 12           
x₁ = 3, x₂ = 4

5. Метод переброски

Метод переброски позволяет решать неприведенные квадратные уравнения через теорему Виета, сводя их к приведенным путем умножения свободного коэффициента на старший или так называемой переброске. В данном случае, найденные корни необходимо разделить на значение коэффициента а.

Пример решения уравнения через метод переброски:

4x² - 17x + 4 = 0
x² - 17x + 16 = 0
x₁ + x₂ = 17  
x₁ ∙ x₂ = 16
x₁ = 16 / 4 = 4, x₂ = 1 / 4 = 0.25

6. Свойства коэффициентов 

Свойства коэффициентов квадратного уравнения позволяют решать квадратные уравнения, не прибегая к методу подбора. Их можно представить в виде двух условий, которые позволяют найти корни уравнения несколькими простыми действиями:

1. Если a + b + c = 0, то x₁ = 1; x₂ = c / a.
2. Если a - b + c = 0, то x₁ = -1; x₂ = -c / a.

Примеры решения уравнений через свойства коэффициентов:

5x² - 7x + 2 = 0
Так как 5 - 7 + 2 = 0, то выполняется свойство (1), а это значит, что корни уравнения x₁ = 1, x₂ = 2 / 5 = 0.4.

2x² + 3x + 1 = 0
Так как 2 - 3 + 1 = 0, то выполняется свойство (2), а это значит, что корни уравнения x₁ = -1, x₂ = -1 / 2 = -0.5.

7. Решение неполных квадратных уравнений

Уравнение ax² + c = 0 решается путем переноса коэффициента c в правую часть уравнения и деления обеих частей на значение коэффициента a. Если частное от деления больше нуля, то корнями уравнения будут являться два значения, которые отличаются только знаками.

Пример решения уравнения вида ax² + c = 0:
3x² - 75 = 0
3x² = 75
x² = 25
x₁ = -5, x₂ = 5

Уравнение ax² + bx = 0 решается путем разложения на множители. В таком случае, вычисление корней уравнения сводится к нахождению значений х, при которых произведение двух получившихся уравнений равно нулю.

Пример решения уравнения вида ax² + bx = 0:
3x² - 12x = 0
3x ∙ (x - 4) = 0
3x = 0, x - 4 = 0
x₁ = 0, x₂ = 4

Уравнение ax² = 0 не требует особенного алгоритма для решения, так как всегда имеет один корень, равный нулю. Если разделить обе части уравнения на значение коэффициента a, то получится выражение x² = 0. Очевидно, что единственным решением данного уравнения будет являться 0.

Пример решения уравнения вида ax² = 0:
8x² = 0
x² = 0
x = 0
