1. Понятие квадратного уравнения

Квадратным уравнением называют уравнение вида:

ax² + bx + c = 0, 

где а, b и c – это числа, называемые коэффициентами и при этом, коэффициент a ≠ 0. 

Коэффициент а называют старшим коэффициентом, коэффициент b – вторым коэффициентом, а коэффициент с – свободным членом. 

Если коэффициент a равен единице, то говорят, что квадратное уравнение является приведенным. 

Квадратные уравнения делятся на полные и неполные. 

Полное квадратное уравнение – это квадратное уравнение, коэффициенты b и c которого отличны от нуля. 
Если же один из этих коэффициентов равен нулю, то такое уравнение называют неполным. 

Существует три вида неполных квадратных уравнений:

ax² + c = 0, при b = 0;
ax² + bx = 0, при c = 0;
ax² = 0, при b = 0 и c = 0.

Решить квадратное уравнение – значит найти его корни, значения переменной x, при которых уравнение превращается в верное равенство или доказать, что их не существует.
